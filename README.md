# Pares

An English and Spanish card matching game made with OpenGL and GLUT.

## Prerequisites

 * Git
 * A C++ compiler
 * OpenGL
 * GLUT
 * CMake

## Installation

To download, build, install, and run:

    git clone https://github.com/cruzantada/pares.git
    cd pares
    cmake .
    make
    ./bin/Pares

## Controls

| Key | Action      |
| --- | ----------- |
| 0-9 | Select card |
| q   | Quit        |
